# qrlink

Страничка, которая показывает QR-код ссылки, переданной в get-параметрах.
Удобно для отображения ссылок в ТВ-браузерах, на сбербоксе итд.

[Пример ссылки](https://vitalets.github.io/qrlink/index.html?url=https%253A%252F%252Fyandex.ru)

```
https://vitalets.github.io/qrlink/index.html?url=https%253A%252F%252Fyandex.ru
```
